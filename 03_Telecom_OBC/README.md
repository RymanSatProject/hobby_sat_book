## 設定の変更

### ブザーの音量を小さくする

ブザーの音量は、`esp32/src/common/config.hpp`中の`CW_VOLUME_PERCENT`で設定されます。音を大きくするには100までの大きな値を、小さくするには0以上の小さな値を設定し、再度ビルド・アップロードしてください。
